Calendar Extension Test

Description
This is a minimal Chrome extension prototype for experimenting with reading a school registration page and eventually auto-populating Google Calendar. Right now, it’s just a “Hello World” popup to confirm the extension loads correctly.

Features (Current)

Popup with a “Click Me” button

Button displays an alert to verify JavaScript is working

Future Goals

Read class schedules from a school registration page

Parse class names, days, times, and locations

Automatically create events in Google Calendar via API

Handle recurring events, conflicts, and multiple calendars

Getting Started

Open Chrome → go to chrome://extensions/

Enable Developer Mode

Click Load unpacked → select this project folder

Click the extension icon → see the popup → click the button for “Hello World”

Technologies Used

HTML, CSS, JavaScript

Chrome Extensions API

Google Calendar API (planned)
